# XIQ - Floorplan uploader
## xiq_floorplan_uploader.py
###
This script will attempt to upload to XIQ any png or jpeg images added to the images folder. You can then choose these files when creating or editing floors

## User Input Data
When running the script the user will be prompted to enter their xiq credentials. If credentials are valid a message will be displayed showing that the script successfully logged in.
> Note: The user must be an administrative user in order to upload floorplans 

## Requirements
The following modules will need to be installed in order for the script to run.
```
requests
```
